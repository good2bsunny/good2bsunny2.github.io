CIS 1300 Networking Concepts

The Networking Concepts course explores components of networks and network designs; communications hardware; network architectures, standards, and protocols; and functions of network operating systems. The outcome of this course includes: understanding network topologies and conepts, TCP/IP Protocol Suite, Cable Media and Network Interface Cards (NICs), virtualization, and Network Operating Systems (NOS).

Noteworthy Project: Virtualization Project

CIS 1200 Intro to Database Systems

The Intro to Database Systems course covers concepts of multi-user database systems, data models, relational database models, entity relationship modeling, normalization and transaction management.
